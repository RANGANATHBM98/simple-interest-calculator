# Simple Interest Calculator

This project explains how to calculate Simple Interest.

## Formula

SI = (P × R × T) / 100

Where:
P = Principal amount  
R = Rate of interest  
T = Time  

## Example

P = 1000  
R = 5%  
T = 2 years  

SI = (1000 × 5 × 2) / 100 = 100

## Conclusion

Simple Interest is easy to calculate using this formula.
